### 2.2 三大主力工具深度评测

#### 🏆 v0.dev（Vercel 出品）— UI质量之王

**定位**：AI驱动的UI组件生成器，专注前端视觉质量

**核心优势**：
- 生成代码质量最高：输出干净的 React + shadcn/ui + Tailwind CSS
- 对话式迭代：反复微调单个组件细节
- 支持图片输入：上传设计稿截图生成对应代码
- 一键部署到 Vercel

**劣势**：仅前端（无数据库/认证/API）、仅React/Next.js生态、免费额度有限

**定价**：Free $0 / Premium $20月 / Premium Pro $50月

#### 🏆 Lovable.dev（原 GPT Engineer）— 全栈MVP之王

**定位**：面向非技术创始人的全栈应用构建器

**核心优势**：
- 深度集成 Supabase：自动生成认证、数据库schema、API
- 文字prompt到可部署应用仅需几分钟
- GitHub同步 + SOC 2 Type II + ISO 27001 认证

**劣势**：仅React+Vite（无Next.js）、信用点消耗快（重度使用$50-100+/月）

**定价**：Free 5 credits/天 / Starter $20月 / Pro $25-50月

**真实项目成本参考**：

| 项目类型 | Lovable | Bolt.new | v0 |
|----------|---------|----------|-----|
| 简单Landing Page | $0-25 | $0-25 | $0-20 |
| SaaS MVP（认证+数据库） | $50-100/月 | $50-75/月+DB | 不适用 |
| 电商站 | $100-200/月 | $75-150/月 | $20-30/月（仅UI）|

#### 🏆 Bolt.new（StackBlitz 出品）— 原型验证之王

**定位**：基于 WebContainers 的全栈AI应用原型工具

**核心优势**：浏览器内运行完整Node.js环境、框架最灵活（React/Next/Vue/Astro/Svelte）、V2增加Bolt Cloud（内置DB/认证/存储）、免费层最慷慨（1M tokens/月）

**劣势**：Token消耗最快、大型项目上下文丢失、无安全认证

**定价**：Free 1M tokens/月 / Pro $25月 10M tokens / Teams $30/用户/月

### 3.3 AI建站通用提示词技巧

1. **先定色彩，再写功能**：AI工具对色彩指令非常敏感，先指定具体hex值
2. **指明字体**：不指定的话AI会用默认无衬线字体，不符合灵性主题
3. **分组件生成**：不要一次生成整个网站，先Hero → Features → Footer逐步做
4. **用图片引导**：先Midjourney生成视觉参考，再上传到v0/Lovable作风格引导
5. **迭代式优化**：对话式逐步精调（"make the gold accents more subtle"）
