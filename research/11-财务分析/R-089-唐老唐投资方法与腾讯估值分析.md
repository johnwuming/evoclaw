## Part 1：唐朝（唐老唐）投资估值方法体系

### 1.1 人物简介

唐朝，网名"唐老唐"，是中国知名的个人投资者和价值投资实践者。2014年至2022年6月公开实盘，年化收益率约26%。著有《价值投资实战手册》《手把手教你读财报》等畅销书。其公众号"唐书房"（现已更名）是中国最活跃的价值投资社区之一。

### 1.2 核心投资理念

唐朝的投资体系核心可以概括为一句话：

> **"三年后以15-25倍市盈率卖出能够赚100%的位置就可以买入，高杠杆企业打七折。"**

这一理念建立在格雷厄姆-多德价值投资框架之上，但做了大幅简化。他的方法本质上是**自由现金流折现模型（DCF）的简化版本**。

#### 哲学基础：不懂不做

唐朝反复强调"认知边界"和"不懂不做"的原则——**只投资自己能看懂的企业**。这不是谦虚，而是风险管理的第一道防线。如果无法理解一家公司的商业模式，就无法判断利润是否可持续，估值就无从谈起。

### 1.3 估值三大前提（门槛条件）

在使用估值公式之前，必须确认企业满足**"三大前提"**：

| 编号 | 前提条件 | 含义 | 验证方法 |
|------|----------|------|----------|
| ① | **利润为真** | 净利润是真实赚到的现金，不是会计调整出来的 | 经营现金流/净利润 ≥ 1；应收账款增速 < 营收增速；自由现金流为正且持续 |
| ② | **利润可持续** | 未来能持续产生同样的利润 | 商业模式有护城河；行业格局稳定；竞争优势可维持 |
| ③ | **维持盈利不需要大量资本投入** | 不需要持续大额资本开支就能维持盈利 | 资本开支/净利润 < 50%；轻资产特征；维护性capex远低于总capex |

**只有三个条件全部满足，才能使用唐朝的估值公式。** 不满足的企业直接排除。

**特别注意：** 增长性资本投入（如AI基础设施投入）和维持性资本投入是两回事。如果企业是为了增长而投入，不影响第三条前提的判定。但需要关注投入能否转化为未来利润。

### 1.4 估值公式详解

#### 第一步：预测三年后净利润

预测目标公司第三年（T+3）的净利润。

**利润口径选择**：唐朝通常使用**Non-IFRS（非国际财务报告准则）净利润**或**扣非净利润**，因为这种口径剔除了：
- 投资性资产的公允价值变动（不可控、不可预测）
- 股权激励费用（非现金支出）
- 一次性损益（出售资产、减值等）

核心逻辑：**估值关注的是可持续的经营性利润，不是波动剧烈的综合收益。**

#### 第二步：确定合理市盈率

**合理市盈率 = 1 / 无风险收益率**

| 无风险收益率范围 | 合理PE取值 | 说明 |
|------------------|------------|------|
| 3%-4% | 25-30倍 | 中国当前主流区间 |
| 4%-5% | 20-25倍 |  |
| 5%-6% | 16-20倍 | |

唐朝通常**取无风险收益率的下限来确定PE**，即偏保守。当前中国10年期国债收益率约1.6%-2.0%，但唐朝认为零利率/极低利率地区并非真的零利率（可投资其他地区资产），实践中一般取**25-30倍**作为合理PE。

**重要：** 唐朝不会因为利率极低就给出50倍以上的PE，他对PE设有上限约束。通常上限为30倍。

#### 第三步：计算三年后合理估值

> **三年后合理估值 = 第3年预计净利润 × 合理市盈率**

高杠杆企业（资产负债率 > 70%）打**七折**。

#### 第四步：确定买点

> **理想买点 = 三年后合理估值 × 50%**

即三年后合理估值打五折。这个50%的安全边际，目的是：
1. 补偿预测错误的风险
2. 估值本身是区间而非精确数字
3. 不占便宜不交易

> **合理买点 = 当年合理估值（当年净利润 × 合理PE）**

唐朝一般从**理想买点开始买入**，到**合理买点**可以满仓。

#### 第五步：确定卖点

卖点的判断标准：

> **一年内卖点 = min（三年后合理估值 × 150%，当年净利润 × 50倍PE）**

即取两者中较低者。

**分批卖出规则**：达到卖点后不一次性卖完，通常分三批：
- 第一批：达到卖点时卖出1/3
- 第二批：再涨约10%卖出1/3
- 第三批：再涨约10%清仓

**卖出分为两种情况**：
1. **估值过高卖出**：按上述卖点公式触发，属于"贵了就卖"
2. **基本面恶化卖出**：发现对企业的判断出现根本性错误（护城河消失、管理层失信等），不问价格立即卖出

### 1.5 仓位管理规则

唐朝对仓位管理有明确的纪律：

| 规则 | 内容 |
|------|------|
| 单票上限 | 通常不超过总仓位的40% |
| 确定性分级 | 按企业"确定性"分等级，确定性越高仓位上限越高 |
| 分批建仓 | 理想买点开始买入，到合理买点逐步加满 |
| 集中持仓 | 通常只持有3-5只股票，不做分散化 |
| 现金管理 | 始终保留一定比例现金应对意外机会 |

### 1.6 选股标准与排雷方法

#### 选股标准
1. **商业模式清晰**：能看懂、能解释清楚怎么赚钱
2. **竞争优势明显**：有护城河（品牌、网络效应、转换成本、规模效应）
3. **ROE持续优秀**：长期ROE > 15%
4. **管理层诚信**：历史信息披露透明，管理层言行一致
5. **行业空间大或格局稳定**：有增长空间或成熟行业但格局清晰

#### 排雷方法（财务分析七项红旗）
1. **应收账款异常增长**：应收增速远超营收增速 → 利润质量存疑
2. **经营现金流持续低于净利润** → 利润可能为纸面利润
3. **频繁的会计政策变更** → 管理层可能在操纵利润
4. **大额商誉** → 并购驱动的增长可能不可持续
5. **关联交易频繁** → 利益输送风险
6. **存货异常增长** → 可能滞销
7. **大存大贷** → 存贷双高可能意味着资金被挪用

### 1.7 对投资性资产的特殊处理

唐朝对持有大量投资性资产的企业（如腾讯），通常采用**分部估值法**：
1. **主营业务**：用上述PE估值法计算
2. **投资性资产**：单独按市值（或折价）计算，不计入PE估值
   - 上市投资：按当前市值 × 折扣系数（通常0.5-0.8）
   - 非上市投资：按最新融资估值或账面价值打折

**总价值 = 主营业务估值 + 投资性资产净值**

### 1.8 唐朝对腾讯的历史估值

唐朝是腾讯的长期持有者，腾讯曾是其最大持仓（占比约40%）。根据公开信息：

- 唐朝从2018年前后开始关注并买入腾讯
- 2021年腾讯高位时接近卖点但未达卖点
- 2022年腾讯大跌至200港币附近时加大买入
- 2024-2025年腾讯反弹后唐朝持续持有
- 唐朝对腾讯使用标准估值法：预测Non-IFRS净利润，按25-30倍PE计算合理估值

### 1.9 估值操作框架总结

```
┌─────────────────────────────────────────────┐
│          唐朝估值操作框架                    │
├─────────────────────────────────────────────┤
│                                             │
│  Step 0: 三大前提检验                       │
│    ├ 利润为真？（现金流验证）               │
│    ├ 利润可持续？（护城河分析）             │
│    └ 维持不需大量投入？（capex分析）        │
│    → 全部通过才继续                         │
│                                             │
│  Step 1: 预测T+3年Non-IFRS净利润            │
│    └ 剔除投资损益、股权激励、一次性损益     │
│                                             │
│  Step 2: 确定合理PE                         │
│    └ = 1/无风险收益率（通常25-30倍）        │
│                                             │
│  Step 3: 三年后合理估值                     │
│    └ = T+3净利润 × 合理PE                  │
│    （高杠杆打7折）                          │
│                                             │
│  Step 4: 买点                               │
│    ├ 理想买点 = 合理估值 × 50%             │
│    └ 合理买点 = 当年净利润 × 合理PE        │
│                                             │
│  Step 5: 卖点                               │
│    └ = min(合理估值×150%, 当年净利×50)      │
│    → 分三批卖出                             │
│                                             │
│  Step 6: 投资性资产单独估值                 │
│    └ 按市值×折扣系数，加到总价值中         │
│                                             │
│  执行规则：                                 │
│    ├ 单票上限40%                            │
│    ├ 持仓3-5只                              │
│    └ 保留现金应对机会                       │
│                                             │
└─────────────────────────────────────────────┘
```

### 2.3 各业务板块增长趋势分析

#### 增值服务（游戏+社交）

**2025年**：增长16%，主要驱动力是自研游戏（《三角洲行动》等）和海外游戏扩张。
**2026Q1**：增速放缓至4%，但毛利率大幅提升至63%（+3ppt），说明高毛利自研游戏占比持续提升。

- 本土游戏流水创新高，但收入确认有时滞
- 国际市场游戏保持稳健增长
- 社交网络增速相对平缓（视频号直播、音乐会员贡献增量）
- **趋势判断**：增速放缓但利润率改善，从"量增"转向"质增"。预计未来三年增速5-10%。

#### 营销服务（广告）

**2025年**：增长19%，增速在所有板块中最高。
**2026Q1**：继续保持20%的高增长。毛利率55%（同比-3ppt），与AI投入成本有关。

- AI驱动的广告推荐模型升级是核心驱动力
- 视频号广告加载率仍有提升空间（当前仍较低）
- 微信搜一搜广告贡献增加
- **趋势判断**：AI+视频号+搜索广告三重驱动，未来2-3年仍可维持15-20%增长。

#### 金融科技及企业服务

**2025年**：增长8%，增速相对较低。
**2026Q1**：增长9%，其中企业服务收入同比增长20%（AI相关需求拉动），毛利率52%（+2ppt）。

- 商业支付和理财服务稳健增长
- 云业务2025年首次实现规模化盈利
- AI相关企业服务需求爆发（云服务、AI推理）
- **趋势判断**：金融科技增长稳健（5-10%），企业服务受AI驱动有望加速（15-20%），整体板块增速约10-12%。

### 2.4 三年净利润预测（2026-2028）

#### 利润口径说明

本预测使用**Non-IFRS归母净利润**，理由：
1. 唐朝估值法关注可持续经营利润
2. IFRS净利润包含投资公允价值波动（腾讯投资组合9,500亿+，波动巨大）
3. Non-IFRS剔除了一次性损益和股权激励费用，更接近经营本质
4. 唐朝本人对腾讯也使用Non-IFRS口径

#### 预测依据详解

1. **2026年Q1实际增速11%（Non-IFRS归母净利润679亿）**，全年按Q1年化约2,716亿。但Q1通常是淡季，下半年有游戏新品+广告旺季+电商支付高峰，全年增速应略高于Q1 → 约11.5%
2. **AI投入加大但变现刚起步**：2026年资本开支预计大幅增长（Q1已增16%至319亿），AI产品（Hy、元宝、CodeBuddy）贡献将逐步释放
3. **营销服务高增长可持续**：AI广告推荐模型升级+视频号商业化+搜索广告
4. **增速逐步放缓的原因**：基数变大（2025年已7,518亿），AI投入短期拉高成本
5. **净利率持续小幅提升**：高毛利业务占比提升+经营杠杆

### 2.5 唐老唐估值法计算

#### Step 0：三大前提检验

| 前提 | 判断 | 依据 |
|------|------|------|
| ① 利润为真？ | ✅ 通过 | 经营现金流/净利润 > 1；Q1自由现金流567亿；总现金5,337亿；现金流验证充分 |
| ② 利润可持续？ | ✅ 通过 | 微信生态（13亿+月活）护城河极强；游戏IP矩阵丰富；AI能力建设加速；广告/支付业务有持续增长动力 |
| ③ 维持不需大量投入？ | ⚠️ 基本通过 | AI时代资本开支增加（Q1资本开支319亿，同比+16%），但腾讯属于轻资产平台型公司。增量投入主要用于AI基础设施建设，属于**增长性投入**而非维持性投入。现有游戏、社交、支付、广告业务的维持性capex较低。 |

**结论：基本符合三大前提，可使用唐朝估值法。**

#### Step 1：预测三年后（2028年）Non-IFRS归母净利润

> **T+3净利润（2028E）= 3,500亿元**（中性情景）

#### Step 2：确定合理市盈率

当前中国10年期国债收益率约1.6%-1.8%。按唐朝的方法：

- 理论PE = 1/1.8% = 约55倍（但唐朝不会用这么高）
- 唐朝实际操作中，在无风险收益率极低时取**25-30倍**
- 腾讯作为平台型互联网公司，确定性较高，取**25倍**（偏保守）

> **合理PE = 25倍**

#### Step 3：计算三年后合理估值（主营业务）

> 三年后合理估值（主营） = 3,500亿 × 25 = **87,500亿元（人民币）**

折算为港币（汇率1人民币 = 1.145港元，2026-05-19央行中间价0.87318的倒数）：
> 87,500亿 × 1.145 = **约100,188亿港元**

#### Step 4：投资性资产单独估值

腾讯持有大量投资性资产，按唐朝的分部估值法：

| 项目 | 估值（亿元） | 说明 |
|------|-------------|------|
| 上市投资公允价值 | 6,727 | 2025年报数据 |
| 折扣系数 | 0.5-0.7 | 非控股、流动性折价 |
| 上市投资估值 | **3,364-4,709** | 取中值约4,000亿 |
| 非上市投资估值 | 约1,500-2,000 | 账面价值打折 |
| **投资性资产合计** | **约5,500-6,500亿元** | **取中值6,000亿元** |

> 投资性资产估值 ≈ 6,000亿元 ≈ 6,870亿港元

**注意：这部分不参与PE估值，直接加到总价值中。**

#### Step 5：计算买点

**总估值 = 主营业务估值 + 投资性资产**

**理想买点**（三年后合理估值×50% + 投资资产）：

> 主营理想买点 = 87,500亿 × 50% = 43,750亿元
> 投资资产 = 6,000亿元
> 总理想买点 = 49,750亿元 ≈ 56,963亿港元

腾讯总股本**91.18亿股**（2026-05-07同花顺F10数据）：
> **理想买点股价 = 56,963 ÷ 91.18 = 约625港元**

**合理买点**（当年净利润×合理PE + 投资资产）：

> 主营合理买点 = 2,895亿 × 25 = 72,375亿元
> 投资资产 = 6,000亿元
> 总合理买点 = 78,375亿元 ≈ 89,739亿港元
> **合理买点股价 = 89,739 ÷ 91.18 = 约984港元**

#### Step 6：计算卖点

> 卖点1：三年后合理估值 × 150% + 投资资产 = 87,500×150% + 6,000 = 137,250亿元 ≈ 157,151亿港元
> 卖点股价 = 157,151 ÷ 91.18 = **约1,724港元**

> 卖点2：当年净利润 × 50倍PE + 投资资产 = 2,895×50 + 6,000 = 150,750亿元 ≈ 172,609亿港元
> 卖点股价 = 172,609 ÷ 91.18 = **约1,893港元**

> **一年内卖点 = min(1,724, 1,893) = 约1,724港元**

### 2.7 估值结论

#### 当前估值状态

当前股价约460港元，**低于理想买点625港元**。

| 指标 | 数值 |
|------|------|
| 当前市值（港币） | 约41,943亿港元（460 × 91.18亿股） |
| 当前市值（人民币） | 约36,662亿元 |
| 距理想买点 | 低约26%（更具吸引力） |
| 距合理买点 | 低约53%（深度折价） |
| 距卖点 | 低约73% |

#### 当前PE验证

按2025年Non-IFRS净利润2,596亿元：
> 当前PE = 36,662亿 ÷ 2,596 ≈ **14.1倍**（不含投资资产）

按2026年预测Non-IFRS净利润2,895亿元：
> 前瞻PE = 36,662亿 ÷ 2,895 ≈ **12.7倍**（不含投资资产）

如果扣除投资资产价值（6,000亿元 ≈ 5,234亿港元）：
> 主营业务市值 = 41,943 - 5,234 = 36,709亿港元 ≈ 32,057亿元
> 主营前瞻PE = 32,057 ÷ 2,895 ≈ **11.1倍**

**无论怎么算，当前PE都远低于唐朝认为的25倍合理PE。**

#### 结论

**按唐老唐的估值框架，腾讯当前处于明显低估区间：**

1. **当前股价（~460港元）低于理想买点（~625港元）**，唐朝估值法认为"理想买点"以下可以积极买入
2. **主营业务前瞻PE仅11-13倍**，远低于唐朝认为的25倍合理PE
3. **安全边际充足**：即使悲观情景（2028年净利润仅3,200亿），理想买点仍约575港元，当前仍有安全边际
4. **三大前提基本满足**：利润为真（现金流验证通过）、可持续（微信护城河极强）、维持不需大量投入（AI投入属增长性投入）
5. **投资性资产额外增厚**：6,000亿元的投资组合价值提供了额外的安全垫

#### 按唐朝执行规则的操作建议

| 当前状态 | 建议 |
|----------|------|
| 股价低于理想买点 | 可开始建仓，按唐朝的分批买入法 |
| 若继续下跌 | 在理想买点以下逐步加仓至满仓 |
| 若上涨至合理买点（~984港元） | 可满仓持有 |
| 若达到卖点（~1,724港元） | 开始分三批卖出 |
| 单票上限 | 不超过总仓位40% |

#### 风险提示

1. **AI投入超预期**：若AI投入大幅增加而变现不及预期，可能压制短期利润（资本开支增速已从个位数升至16%+）
2. **宏观环境**：消费疲软可能影响广告和支付收入
3. **游戏监管**：政策变化可能影响游戏收入
4. **投资组合波动**：9,500亿+的投资组合公允价值波动巨大，可能影响IFRS利润和市场情绪
5. **估值法局限性**：唐朝估值法的25倍PE假设和50%安全边际是经验值，不是精确计算
6. **汇率风险**：人民币/港元汇率波动影响港元计价的估值（当前约1:1.145，若人民币贬值则买点对应的港元价格更低）
7. **增速放缓**：Q1营收增速已降至9%，若未来持续个位数增长，25倍PE的合理性需要重新评估
