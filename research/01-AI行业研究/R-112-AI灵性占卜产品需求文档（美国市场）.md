# PRD: 「MysticMirror」— AI灵性占卜产品需求文档

> **产品代号**: MysticMirror | **目标市场**: 美国市场（面向英语用户）
> **版本**: v1.0 MVP | **日期**: 2026-06-23
> **调研依据**: R-110（美国塔罗牌市场调研）+ R-111（中国玄学出海机会调研）
> **开发者**: 一人（AI辅助开发）

---

## 一、产品定位

### 1.1 一句话定位

> **"Your birth, decoded by East and West — an AI-powered spiritual companion."**
> （你的出生信息，东西方双视角AI解读——你的24/7灵性陪伴）

### 1.2 核心差异化

| 维度 | 竞品现状 | MysticMirror |
|------|----------|-------------|
| 解读体系 | 单一西方体系（塔罗/星座） | **东西方融合**：塔罗+五行+八字+易经 |
| AI深度 | 模板化内容，"冷读感"强 | **AI个性化深度解读**，基于用户完整出生信息 |
| 文化角度 | 无差异化 | **东方玄学作为护城河**，风水/八字赛道几乎空白 |
| 产品形态 | App为主 | **Web优先**（规避App Store占卜审核风险） |

### 1.3 为什么可行？

- 美国玄学/灵性App市场$14亿+（北美占全球37%），CAGR 20-26%
- 30%美国人每年至少参与一次占星/塔罗（Pew Research）
- **头部竞品（Co-Star、Sanctuary、The Pattern）均未深度整合生成式AI**——蓝海窗口
- 中国玄学元素中，风水认知度极高（TikTok #fengshui 34亿播放），但**没有头部App**
- AI是"文化翻译"的关键：将晦涩的八字/易经转化为西方用户能理解的个性化建议

## 二、用户痛点与需求分析

### 2.1 核心痛点矩阵

| # | 痛点 | 严重度 | 竞品现状 | MysticMirror 解法 |
|---|------|--------|----------|------------------|
| 1 | **解读千篇一律**（冷读感） | ⭐⭐⭐⭐⭐ | Co-Star/塔罗App都是模板内容，换个人解读一样 | AI基于出生时间+性格+当前问题生成**个性化深度解读** |
| 2 | **真人解读太贵** | ⭐⭐⭐⭐ | Sanctuary真人$4.99-19.99/分钟 | AI解读$0.03-0.10/次（API成本），用户付费$1.99-4.99/次 |
| 3 | **想要更多体系视角** | ⭐⭐⭐⭐ | 用户看完星座还想看八字/风水，要切换多个App | **一个产品提供东西方双视角** |
| 4 | **入门门槛高**（尤其东方玄学） | ⭐⭐⭐⭐ | 八字术语（天干地支/十神/神煞）对西方人如同天书 | AI用通俗英语"翻译"，用MBTI式框架包装五行人格 |
| 5 | **缺乏情绪安全空间** | ⭐⭐⭐ | 16-25岁年轻人需要非评判性的情绪导航工具 | 产品定位为"自我探索与反思"，不是"算命预测" |
| 6 | **社交分享需求** | ⭐⭐⭐ | 抽牌结果想分享给朋友/社交圈 | 精美的分享卡片，TikTok/Instagram一键分享 |
| 7 | **每日仪式感缺失** | ⭐⭐⭐ | 偶尔用一次就忘 | 每日运势推送+每日一牌，养成习惯 |

### 2.2 需求层次（马斯洛映射）

```
🎯 底层需求：娱乐、好奇心驱动的占卜体验
    → 免费每日运势、每日一牌
    
🎯 中层需求：自我认知与情绪导航
    → 深度塔罗解读、五行人格分析、情绪日记
    
🎯 高层需求：人生决策支持与关系洞察
    → AI对话式解读、兼容性分析、年度运势
    
🎯 社交需求：归属感与身份表达
    → 分享卡片、生肖标签、社区内容
```

### 4.4 暂不做的功能（明确排除）

| 功能 | 排除理由 |

| 紫微斗数 | 西方认知度接近零，唯一英文App仅62次下载1星，教育成本过高 |

| 算命/预测（fortune-telling定位） | 规避App Store审核风险和文化敏感性争议，定位为"自我探索工具" |

## 五、产品设计与选型

### 5.1 设计风格定义

**关键词**: 暗色星空 + 东方神秘美学 + 现代极简

| 设计维度 | 方案 | 参考 |
|----------|------|------|
| **色调** | 深蓝黑底（#0a0a1a）+ 金色accent（#d4a574）+ 紫罗兰渐变（#7c3aed → #a78bfa） | 现有国内塔罗MVP风格延续 |
| **字体** | 英文：Playfair Display（标题）+ Inter（正文）；中文元素用Noto Serif SC | 优雅+可读性平衡 |
| **塔罗牌设计** | 保持现有78张牌设计，可考虑在牌背加入东方纹样（云纹/八卦纹） | 差异化细节 |
| **五行人格UI** | 用五种元素色彩（木绿/火红/土黄/金白/水蓝）做视觉标识，配合图标系统 | MBTI式直觉化设计 |
| **动画** | 抽牌翻转动画（3D CSS）、星空粒子背景、解读文字流式打字效果 | 沉浸感 |
| **分享卡片** | 16:9竖版卡片，深色背景+塔罗牌图+关键解读文字+产品水印 | TikTok/IG Stories友好 |
| **响应式** | Mobile-first，适配375px-428px宽度；桌面端双栏布局 | 80%+用户来自移动端 |

### 5.2 东西方融合设计原则

1. **UI不突兀融合**：塔罗/星座用西方视觉语言，五行/八字/风水用东方视觉语言，但整体色调统一
2. **渐进式东方元素**：免费版偏西方（塔罗+星座），付费版解锁东方元素（五行+易经）——降低入门门槛
3. **文化尊重**：东方元素不做"exotic fetishization"，以知识性+工具性呈现，避免刻板印象

### 5.3 关键页面清单

| 页面 | 核心元素 |
|------|----------|
| 首页（Landing） | 品牌标语 + 产品特色（3个icon）+ 立即体验CTA + 社会证明 |
| 出生信息采集 | 出生日期/时间/地点（精确到分钟）+ 隐私承诺 |
| 五行人格报告 | 五行占比可视化（雷达图/环形图）+ 性格描述 + 东方元素图标 |
| 塔罗占卜页 | 牌阵选择 → 洗牌动画 → 抽牌（点击翻转）→ AI流式解读 |
| 每日运势 | 西方星座+五行双栏 + 一句话洞察 + 分享按钮 |
| 定价页 | 3个plan对比（Free/Pro/Premium）+ 功能对比表 |
| 分享结果页 | 精美卡片 + 下载/分享/TikTok链接 |

## 六、技术架构选型

### 6.1 技术栈（一人开发者友好）

| 层 | 选型 | 理由 |
|----|------|------|
| **前端框架** | Next.js 14 (App Router) | SEO友好（利于ASO/SEO获客），React生态成熟，Vercel一键部署 |
| **样式** | Tailwind CSS + Framer Motion | 快速开发精美动画，响应式设计 |
| **塔罗牌渲染** | CSS 3D Transform + SVG | 轻量级翻牌动画，无需Canvas/Three.js |
| **AI后端** | Next.js API Routes → LLM API | 免服务器运维，前端统一技术栈 |
| **AI模型** | Claude Haiku / GPT-4o-mini（解读）+ 结构化prompt | 成本低（$0.03-0.10/次），质量足够 |
| **塔罗牌库** | 现有78张牌素材（Rider-Waite风格自定义版） | 已有资产，复用 |
| **支付** | Stripe Checkout（Web） | 全球支持，一键接入，订阅管理成熟 |
| **邮件推送** | Resend / SendGrid | 每日运势邮件推送 |
| **分析** | PostHog（开源） | 免费额度足够MVP，事件分析+漏斗 |
| **部署** | Vercel + Cloudflare CDN | 免费起步，自动扩容 |
| **域名** | mysticmirror.app / eastwestoracle.com（待定） | 品牌感+SEO |

### 6.2 AI Prompt架构

```
System: 你是一位融合东西方玄学智慧的AI占卜师。
       用户出生信息：{birth_info} → 五行：{five_elements}，星座：{western_sign}
       当前占卜类型：{reading_type}
       用户问题/情绪：{question_or_mood}
       
Rules:
- 用温暖、有洞察力的英语，避免术语堆砌
- 东方概念用比喻和日常语言"翻译"（如：木元素=创造力和生长能量）
- 给出可行动的建议，不做确定性预测
- 引用对应的东西方概念做对照解读
- 语气：像一位智慧的朋友，不是神秘的先知
```

### 6.3 八字/五行计算

- **出生时间 → 天干地支 → 五行**：需要农历转换算法 + 天干地支计算库
- 开源方案：`lunar-javascript`（npm），支持农历、节气、天干地支
- 五行权重计算：年柱/月柱/日柱/时柱 → 五行统计 → 生成"五行人格画像"

## 十、风险与应对

| 风险 | 严重度 | 应对策略 |
|------|--------|----------|
| **App Store占卜审核风险** | 🔴 高 | Web优先，App后置；产品定位"自我探索工具"而非"fortune-telling" |
| **文化挪用争议** | 🟡 中 | 在叙事中尊重文化源头，有文化顾问/AI审核机制，避免刻板印象 |
| **AI解读质量不稳定** | 🟡 中 | 结构化Prompt+Few-shot示例+输出审核模板；用户反馈迭代 |
| **竞品快速跟进** | 🟡 中 | 东方玄学是天然护城河（竞品要理解八字/五行体系需要时间） |
| **用户付费意愿低于预期** | 🟡 中 | Freemium验证 → 调价测试 → 增加单次购买选项 |
| **中国玄学教育成本高** | 🟢 低 | AI翻译解决，渐进式东方元素（免费版西方、付费版东方） |
| **Stripe支付合规（中国开发者）** | 🟢 低 | 需单独调研Stripe Atlas / Paddle 合规方案 |
