# R-038 猪周期数据源调研

> 调研日期：2026-04-07
> 目标：为猪周期看板找到能繁母猪存栏、仔猪价格等关键指标的免费/低成本数据源

---

## 一、能繁母猪存栏量

### 1.1 五部门联合发布平台（推荐首选）

全国畜牧总站（nahs.org.cn/jcyj/）发布月度能繁母猪存栏数据，来源为农业农村部、国家发改委、商务部、海关总署、国家统计局五部门联合机制。

- **URL**: https://www.nahs.org.cn/jcyj/
- **频率**: 月度
- **时间跨度**: 2021年至今
- **免费/付费**: 免费
- **采集方式**: 需爬虫抓取网页表格（无官方API）
- **数据示例**: 2025年12月末3961万头，正常保有量3900万头

### 1.2 国家统计局

- **URL**: https://data.stats.gov.cn/
- **频率**: 季度末（3/6/9/12月末），非季度末月份由农业农村部定点监测环比推算
- **采集方式**: 网页查询，可能有AJAX接口可通过抓包发现
- **免费/付费**: 免费

### 1.3 大畜牧网（第三方整理）

- **URL**: https://www.dxumu.com/
- **频率**: 月度，每月汇总五部门数据
- **采集方式**: 爬虫
- **免费/付费**: 免费
- **备注**: 结构化程度较高，适合爬取

### 1.4 农业农村部官网

- **URL**: https://www.moa.gov.cn/ztzl/szcpxx/jdsj/
- **采集方式**: 需JS渲染，采集难度较高

## 二、仔猪价格

### 2.1 AKShare `futures_spot_sys_df`（推荐首选）

AKShare期货数据接口 `futures_spot_sys_df` 返回字段包含"仔猪价格"（float64），数据来源标注为农业农村部等官方渠道。

- **频率**: 日度/周度
- **采集方式**: Python API，一行代码
- **免费/付费**: 免费
- **备注**: 需验证该"仔猪价格"是否为断奶仔猪价格；AKShare文档: https://akshare.akfamily.xyz/data/futures/futures.html

### 2.2 农业农村部畜牧兽医局监测预警

- **URL**: https://xmsyj.moa.gov.cn/jcyj/
- **频率**: 周度（全国500个农村集贸市场）
- **内容**: 仔猪平均价格（元/公斤），含周环比、同比
- **采集方式**: 爬虫
- **免费/付费**: 免费
- **状态**: 2026年3月仍在更新

### 2.3 AKShare 生猪供应维度接口

知乎文章（zhuanlan.zhihu.com/p/688229417）介绍AKShare新增接口，涵盖猪肉批发价、育肥猪、生猪产能、猪粮比价等数据。

### 2.4 搜猪网/中国养猪网

- **URL**: soozhu.com / zhuwang.cc
- **频率**: 日度
- **采集方式**: 爬虫（无公开API）
- **免费/付费**: 部分免费

---

## 三、生猪期货价格

### 3.1 AKShare `futures_hog_spot`（推荐）

- **返回内容**: 生猪期货合约数据（如LH2111、LH2201开盘价/收盘价），非现货价格
- **频率**: 日度（交易日）
- **时间跨度**: 2021-01-08至今
- **采集方式**: Python API
- **免费/付费**: 免费
- **文档**: https://akshare.akfamily.xyz/data/futures/futures.html

### 3.2 大连商品交易所官方API（2025年新增）

大商所于2025年上线免费API服务，面向注册用户开放，支持日/周/月行情数据、统计信息等。

- **申请入口**: 门户网站"数据服务→数据工具→API服务"
- **频率**: 日度
- **免费/付费**: 免费（需注册）
- **来源**: http://www.dce.com.cn/dce/content/2025/wm/18625789.html

### 3.3 AKShare现货价格接口

- `futures_spot_price_previous` — 具体交易日大宗商品现货价格及基差
- `futures_spot_price_daily` — 一段交易日现货价格
- `futures_spot_price` — 大宗商品现货价格
- 数据来源: 生意社（2011年至今）

## 五、推荐方案

| 看板指标 | 推荐数据源 | 实现方式 |
|----------|-----------|----------|
| **能繁母猪存栏** | 五部门联合平台(nahs.org.cn) | 定时爬虫，月度更新 |
| **仔猪价格** | AKShare `futures_spot_sys_df` | Python API，周度/日度 |
| **生猪期货价格** | AKShare `futures_hog_spot` | Python API，日度 |
| **生猪现货价格** | AKShare `futures_spot_price` | Python API，日度 |
| **猪粮比价** | AKShare 生猪供应维度接口 | Python API |
| **屠宰量** | 商务部网站 | 爬虫，周度 |
| **猪饲料产量** | 中国饲料工业协会 | 爬虫，月度 |
| **猪肉产量/消费** | 国家统计局年度 + 推算 | 年度手动更新 |

**实施优先级**:
1. AKShare现有接口（期货价格、现货价格、猪粮比、仔猪价格）— 零成本，立即可用
2. nahs.org.cn爬虫（能繁母猪存栏、出栏量）— 需开发爬虫，月度更新
3. 商务部屠宰量爬虫 — 需开发爬虫，周度更新
4. 大商所官方API注册 — 补充期货数据

---

## 六、风险与缺口

1. **能繁母猪存栏无官方API** — 所有数据源均为网页/表格形式，需爬虫维护，页面改版可能导致采集失败
2. **AKShare仔猪价格接口需验证** — `futures_spot_sys_df`中的"仔猪价格"字段是否等同于断奶仔猪价格，需实际调用确认
3. **大畜牧网/搜猪网为非官方来源** — 数据准确性依赖其更新及时性
4. **大商所API为2025年新上线** — 接口稳定性有待观察，需注册申请
