# 猪周期数据可视化看板 产品文档

## 产品概述
面向个人投资者的猪周期专业分析看板，一屏展示猪周期全链条核心指标（猪价、能繁母猪、仔猪价格、养殖利润），帮助用户快速判断周期位置和投资方向。基于 R-037 设计方案。

## 技术栈
- 后端：FastAPI + SQLite + AKShare
- 前端：HTML + Vanilla JS + ECharts（CDN）
- 部署：systemd + Nginx 反代

## 技术决策
- 选择 FastAPI + ECharts 而非 Dash + Plotly：内存占用更低（~50MB），图表交互质量更高（金融终端级 dataZoom/联动），无状态 API 更适合多用户
- 单页面设计（不做多页导航）：信息密度优先，参考专业金融终端
- SQLite 存储：数据量极小（<1MB），零运维
- AKShare 实际接口与文档不符：`futures_hog_info` 已不存在，使用 `spot_hog_year_trend_soozhu` 等替代；能繁母猪暂用模拟数据
- 传导图使用 ECharts graph 而非桑基图：graph 支持拖拽、缩放，更适合展示因果链关系
- 股票数据使用确定性模拟（基于日期哈希）：同一天内数据一致，避免每次刷新变化

## 已知限制
- 能繁母猪存栏数据为模拟数据（AKShare 无可靠月度数据源），需接入国家统计局数据
- 仔猪价格为模拟数据（AKShare 无直接接口）
- 期货 LH 连续合约数据未接入
- 出栏体重、屠宰量等需付费数据源未纳入
- 投资标的股票数据为模拟数据，AKShare 虽有 `stock_zh_a_hist` 接口但需实时调用，当前使用确定性模拟
