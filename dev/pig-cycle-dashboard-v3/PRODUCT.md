# 猪周期数据看板 v3 产品文档

## 产品概述

猪周期数据可视化看板 — 基于深色金融主题的中文数据看板，实时展示生猪价格、饲料价格、猪粮比、能繁母猪存栏等核心指标，帮助养殖从业者把握猪周期规律。

## 技术决策

- **Dash + Plotly**：Python 生态，图表交互能力强，部署简单
- **AKShare**：开源中国金融数据接口，覆盖生猪期货/现货、玉米、豆粕价格
- **SQLite**：本地缓存，零配置，适合单机部署
- **红涨绿跌**：遵循 A 股惯例，与国际惯例相反
- **深色主题**：专业金融终端风格，长时间观看不疲劳
