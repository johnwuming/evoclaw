# 猪周期数据看板 产品文档

## 产品概述
基于 Dash + Plotly 的猪周期数据自动看板，7×24 运行，自动从 AKShare 采集猪价/猪粮比/饲料数据，提供归一化叠加图、猪价走势、能繁母猪占位、猪粮比、周期仪表盘、投资标的共 6 个页面。

## 技术决策
- Dash + Plotly：回调机制省资源，Plotly 交互性最佳（R-032 推荐）
- SQLite：数据量极小（年增 52 周度行），单文件零配置
- 期货猪价作为日度主数据源（`futures_hog_core`），搜猪网作为现货补充
- 猪粮比计算：hog_price / corn_price
- 无 SSE/WebSocket，采用 dcc.Interval 轮询（5分钟间隔）
