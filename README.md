## 目录结构

```
evolving-claw-repo/
├── research/              # 研究报告（按分类子目录组织）
│   ├── 01-研发团队架构/
│   ├── 02-AI技术调研/
│   ├── 03-产品功能/
│   ├── 04-变现与增长/
│   ├── 05-量化投资/

│   ├── 07-其他/
│   └── internal/           # 研究过程文件

```

## 命名规范

| 前缀 | 来源 | 格式 | 示例 |
|------|------|------|------|
| R-xxx | 研究团队 | R-xxx-中文标题.md | R-001-深度研究团队配置.md |
| M-xxx | Main agent | M-xxx-中文标题.md | M-001-频道插件兼容性Bug.md |

- xxx：三位递增编号，不复用
- 中文标题：简洁概括，≤10字，专业术语保留英文

## 自动同步

本地 `~/.openclaw/` 下各 workspace 的产出物通过 auto-sync.sh 自动同步到本仓库。

**同步规则：**
- `shared/results/<分类>/` 整体镜像到 `research/<分类>/`（保留目录结构）
- `workspace-dev/<项目>/` 同步到 `dev/<项目>/`
- `workspace-research/research/` 同步到 `research/internal/`

**注意：** 更新日志需要手动维护（见上方格式）。新增报告时，在顶部 `### 2026-04-06` 下方追加即可。
